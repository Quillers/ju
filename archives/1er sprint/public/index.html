Je suis le module Message

Mes objectifs sont :

  • Créer l'architecture permettant l'affichage des "noms du sujet", "Messages" et du formulaire de saisie. Objet 'Topic' => ok
  • La validation de la saisie d'un message doit envoyer le message au serveur qui affiche en retour le message sur l'affichage principal. Faire une méthode dans 'Topic'
  • Créer un format d'affichage pour les messages, également sous forme de formulaire. Pour cela faire un objet "Message"
